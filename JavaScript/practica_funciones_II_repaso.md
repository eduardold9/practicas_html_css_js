# Ejercicios Básicos de JavaScript

> Todos los ejercicios están pensados para practicar conceptos básicos de JavaScript.
> No es necesario resolverlos con HTML ni CSS, salvo que se indique lo contrario.
> Puedes probarlos directamente en la consola del navegador o en un archivo `.js`.

---

# Bloque 1 - Ejercicios con `console.log()`

## Ejercicio 1

Muestra por consola el siguiente mensaje:

```javascript
"Estoy aprendiendo JavaScript paso a paso"
```

---

## Ejercicio 2


## Ejercicio 4

---

## Ejercicio 5

Muestra por consola dos mensajes diferentes, uno debajo del otro:

```javascript

"Fin del programa"
```

# Bloque 3 - Ejercicios con condicionales

## Ejercicio 16

Crea una variable externa llamada `stockDisponible`.

Si el valor es mayor que `0`, muestra por consola:

```javascript
"Producto disponible"
```

```javascript
"Producto agotado"
```

