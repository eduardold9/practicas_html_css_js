# Actividad 5: Añadiendo un formulario de contacto básico

Modifica la página HTML que has construido hasta ahora y añade un formulario de contacto sencillo.

Realiza los siguientes cambios:

1. Añade una etiqueta `<form>` en la zona de contacto o en una nueva sección de la página.

2. Configura los atributos `action` y `method` en el formulario.

3. Incluye al menos dos campos de entrada, uno de tipo texto y otro de tipo email.

4. Añade una etiqueta `<label>` asociada correctamente a cada campo mediante `for` e `id`.

5. Incorpora un área de texto para que la persona usuaria pueda escribir un mensaje.

6. Añade un botón de envío con el texto que consideres adecuado.

7. Revisa que la estructura del formulario sea clara y fácil de entender dentro del conjunto de la página.
